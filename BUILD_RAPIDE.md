# ⚡ Build Rapide pour Hostinger

## 🚀 Commandes simples

### 1. Générer le build
```bash
npm run build:hostinger
```

### 2. Le dossier `out` est prêt !
Le dossier `out` contient tout ce dont vous avez besoin pour déployer sur Hostinger.

### 3. Déployer sur Hostinger
1. Connectez-vous à **hPanel Hostinger**
2. Allez dans **File Manager**
3. Ouvrez le dossier **`public_html`**
4. **Uploadez TOUT le contenu du dossier `out`**

✅ **C'est tout !** Votre site est en ligne.
